# DIO-Ecommerce
Base de dados de dados fictícios de um e-commerce criada do zero usando SQL

Modelagem do projeto lógico de banco de dados para o cenário de e-commerce,com chave primária e estrangeira, assim como as constraints presentes no cenário modelado. 
Realize a criação do Script SQL para criação do esquema do banco de dados e queries para consulta e análise dos dados.
